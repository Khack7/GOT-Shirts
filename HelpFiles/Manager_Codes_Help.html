Table of Contents

To see created codes along with their respective information, click the dropdown and select one of the codes

To activate the selected code, simply click 'Activate' to make it usable in the shop

Likewise click 'Deactivate' to deactivate it and make it unsuable in the shop

To create a new code click 'Create New Code' and input the code in the textbox adjacent to the text "Code:"

Afterwards input the percentage amount the code will reduce a purchase by in the textbox adjacent to the text "Percent Off:"

Once both text fields are filled click 'Create', or to cancel click 'Cancel'

Click 'Return to go back to the Manager Hub'
